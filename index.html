TeeClap - Wear Your Mood

Completed On June 8, 2018

TeeClap.com is an online T-Shirt store, created in Shopify.

This is an apparel-based eCommerce website designed in Shopify. For this website, we have used Shopify as a hosting and CMS platform. This website was completed on June 8th, 2018 within a span of 70 days.

The main challenge in this website was to manage data using an Excel sheet containing all the details of the products. This problem was catered to by generating a custom-designed Excel which was data-populated, and with the help of excel formulas the same was exported into shopify-based format.

Apart from graphics and banner images are also created for different product categories and pages within this website.
